# 租赁管理系统

## 项目结构
- `src/` - 源代码
  - `components/` - 公共组件
  - `views/` - 页面视图
  - `styles/` - 样式文件
  - `utils/` - 工具函数
  - `api/` - API 接口
- `public/` - 静态资源
- `.claude/skills/` - Claude Skill
- `prompts/` - 提示词模板
- `docs/` - 项目文档

## 开发规范
- 使用 Vue 3 + TypeScript
- 遵循设计系统规范
- 组件命名：PascalCase
- 文件命名：kebab-case
